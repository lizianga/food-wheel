# 🍽️ 今天吃什么 - 智能菜单转盘

<div align="center">

**不知道吃什么？让转盘帮你决定！**

一个可爱的随机选择转盘，支持自定义菜品和分类

[在线体验](https://lizianga.github.io/food-wheel/) · 简单易用 · 数据本地存储

</div>

---

## ✨ 功能特性

- 🎡 **随机转盘** - 点击按钮，转盘帮你选择今天吃什么
- 🍽️ **智能推荐** - 根据时间智能推荐早餐/午餐/晚餐
- 📂 **分类管理** - 支持多种分类（早餐、午餐、晚餐、小食、饮品）
- ➕ **自定义菜品** - 可以添加自己爱吃的菜品
- 🎨 **自定义图标** - 为每个菜品选择可爱的 emoji 图标
- 🗑️ **删除/恢复** - 不想要的菜品可以删除，也可以恢复
- 💾 **本地存储** - 所有数据保存在浏览器本地，无需登录
- 🎀 **可爱界面** - 马卡龙配色，圆润设计，带动画效果

## 📖 使用说明

### 基础使用

1. **选择分类**：点击顶部的分类按钮（早餐、午餐、晚餐等）
2. **开始转动**：点击"🎲 开始转动"按钮
3. **查看结果**：转盘停止后会弹出选中的菜品

### 自定义菜品

1. 点击 **"⚙️ 管理菜品"** 按钮
2. 在表单中输入菜品名称
3. 选择一个分类
4. 从表情选择器中选择一个图标
5. 点击 **"➕ 添加菜品"** 按钮

### 管理分类

1. 点击 **"📂 管理分类"** 按钮
2. 添加新分类：输入名称并选择图标
3. 可以重命名或删除自定义分类

### 修改/删除菜品

在"菜品管理"中：
- 🎨 换图标：修改菜品的 emoji 图标
- ✏️ 改名：修改菜品名称
- 🗑️ 删除：删除不需要的菜品
- ↻ 恢复：恢复已删除的默认菜品
